[ きづまり 傘の下の環境音楽 ] website




Medium　　Website

year 　　November2025

software　　photoshop / Visual Studio Code


本作は、音楽グループ kidumari によるイベント 「きづまりの傘の下の環境音楽」のために制作したWebサイトです。

本イベントにおける最大の課題は、 多くの楽曲が観客にとって「ライブで初めて聴く曲」である点にありました。

そこで、来場前から作品に触れられるよう、 事前体験を見込んだWebサイトを制作しました。

サイト内には、各楽曲の目次、歌詞、制作背景、作曲者のコメントなどを掲載し、 観客が楽曲や文脈を理解した上で会場に臨める構成としています。

また、マイク機能を用いて会場で再生されている楽曲を認識し、 該当する楽曲ページへ自動的に遷移する仕組みを実装しました。

これにより、来場者のイベントに対しての没入感をより高めることができました。
