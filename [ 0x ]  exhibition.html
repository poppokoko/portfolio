[ Untitled ] window


Medium　　 window

year 　　december2025

software　　PLA / acrylic / Unity



「精神的な都市空間の中で、個を再構築するための窓」をテーマにした作品。

立体造形した窓枠の内部には、実際の天候データと連動するインタラクティブなCGアニメーションを表示している。

鑑賞者は窓の向こうに広がるもう一つの風景を覗き込むことで、

日常から離れた視点を獲得し、自身と世界との関係性を見つめ直す。







[ Untitled ] installation


Medium　　 installation

year 　　december2025

software　　Raspberry Pi/ acrylic / water



水面に与えた微弱な振動は、絶えず変化する光の模様となって空間全体へ広がる。

その光景は、天井付近で静かに沸き立つ雲や霧のようにも見え、
鑑賞者に重力や空間の感覚の揺らぎをもたらす。

本作品は、普段見過ごされるほど小さな現象を拡張し、
空間そのものが呼吸しているかのような体験を生み出すことを目指した。
